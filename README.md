# Typing Speed Test (Java)

This is a simple Java project where you can check your typing speed and accuracy in the terminal.

## What It Does

- Shows you a bunch of words from a file
- You type them one by one
- It checks how fast and how accurately you typed
- Gives you results at the end

## What You Need

- Java installed on your computer
- A file called `words.txt` with some words in it (space-separated)

## How To Run It

1. Open your terminal or command prompt
2. Go to the folder where this project is
3. Compile the code:
   ```bash
   javac TypingSpeedTest.java
